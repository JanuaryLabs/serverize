This applies on projects that use a bundler like [esbuild](https://esbuild.github.io/), [webpack](https://webpack.js.org/), [rollup](https://rollupjs.org/), [parcel](https://parceljs.org/), or [vite](https://vitejs.dev/).


<details open>
<summary>Dockerfile</summary>

```dockerfile title="Dockerfile"
FROM node:alpine

WORKDIR /app

COPY ./dist .

# Expose the port your app runs on
EXPOSE 3000

# Start the application
CMD ["node", "index.js"]
```

</details>

<details>
<summary>.dockerignore</summary>

```.dockerignore title=".dockerignore"

**/node_modules/
**/.git
**/README.md
**/LICENSE
**/.vscode
**/npm-debug.log
**/coverage
**/.env
**/.editorconfig
**/.aws
**/dist

```

</details>



_In watch mode, make sure to have the bundler in watch mode too._
